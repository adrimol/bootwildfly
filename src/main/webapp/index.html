OPERACIONES DISPONIBLES

Listado de medicos: /medico

Buscar por ID medico: /medico/ID
ID: Identificador del registro

Buscar por palabra clave: /medico/buscar/KEYWORD
KEYWORD: Palabra clave que puede ser nombre, apellido o especialidad.

Crear registro: /medico/crear/REGISTRO
REGISTRO: ID-Nombres-Apellidos-Edad-Especialidad
Ejemplo: /medico/crear/6-Pedro-Perez-45-Traumatologia

Actualizar registro: /medico/actualizar/REGISTRO
REGISTRO: ID-Nombres-Apellidos-Edad-Especialidad
Ejemplo: /medico/actualizar/6-Pedro-Perez-36-Gastroenterologia

Borrar por ID medico: /medico/borrar/ID
ID: Identificador del registro
